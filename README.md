# cable-assembling

A project to bundle requirements for cable assembling.

1. Cut length of cable + 5cm±2cm. Ex 100cm cable: 105±2cm.
2. Remove out side covering material 2cm±1cm at both end.
3. Put 2cm cover tube at both end.
4. Remove inner covering material 2mm.
5. Put contacts to all 8 wires.
6. Inject contacts to housing.
7. Make shrink cover tube by heating.
